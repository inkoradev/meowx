To-Do List Commands

Here are some useful commands for your To-Do List:

!add_task <task_name>

Add a new task with the current date as the due date. Example: !add_task Buy groceries

!mark_done <task_number>

Mark a task as completed. Example: !mark_done 2

!edit_task <task_number> <new_task_name>

Edit an existing task. Example: !edit_task 1 Finish homework

!change_date

Changes the due daye. Example: !change_date

!show_tasks

Show tasks in a table. Example: !show_tasks

!help

redirects you to the Documentation page. Example: !help

!signout

Sign out from the website, but your data will be stored until you close the tab. Example: !signout
